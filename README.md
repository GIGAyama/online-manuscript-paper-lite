# **オンライン原稿用紙 Lite 📝**

**サーバー不要、設定不要、配布するだけ。**

GIGAスクール構想下の教育現場のために設計された、ブラウザだけで動作する軽量な縦書き原稿用紙アプリです。

## **🌟 概要**

「オンライン原稿用紙 Lite」は、Google Apps Script (GAS) やデータベースを一切使用しない、クライアントサイド完結型のWebアプリケーションです。

index.html という1つのファイルをブラウザで開くだけで、誰でもすぐに原稿用紙を使った作文活動を開始できます。

### **なぜ Lite なのか？**

* **導入が圧倒的に簡単**: 先生がファイルを生徒に配るか、Webサーバーに置くだけで導入完了。  
* **プライバシー**: 入力されたデータはサーバーに送信されず、すべて端末内（ブラウザとローカルファイル）で完結します。  
* **メンテナンスフリー**: データベースの管理やAPIの制限を気にする必要がありません。

## **✨ 機能**

1. **縦書きリアルタイムプレビュー**  
   * 左に入力すると、右の原稿用紙に即座に縦書きで反映されます。  
   * 句読点の「ぶら下げ」や「追い出し」などの禁則処理も自動で行います。  
2. **用紙設定の切り替え**  
   * **高学年用**: 20字 × 20行  
   * **低学年用**: 15字 × 16行  
   * ワンクリックで切り替え可能です。  
3. **データの保存と読み込み**  
   * **自動保存**: ブラウザを閉じても、次回アクセス時に続きから再開できます（LocalStorage使用）。  
   * **ファイル保存**: 作成した作文をテキストファイル（.txt）としてダウンロードできます。  
   * **読み込み**: 保存したテキストファイルを読み込んで再編集できます。  
4. **印刷・PDF化**  
   * ブラウザの印刷機能（Ctrl \+ P）を使って、A4横向きのきれいな原稿用紙レイアウトで印刷・PDF保存ができます。

## **🚀 導入方法（先生・管理者向け）**

以下のいずれかの方法で利用を開始できます。

### **方法A: GitHub Pages で公開する（推奨）**

1. このリポジトリをForkするか、index.html をダウンロードして自分のリポジトリに配置します。  
2. GitHubリポジトリの Settings \> Pages から、公開元（Source）を main ブランチに設定します。  
3. 発行されたURLを児童・生徒に共有してください。

### **方法B: ファイルとして配布する**

1. index.html をダウンロードします。  
2. Google ClassroomやMicrosoft Teams、USBメモリなどを通じて、児童・生徒の端末にファイルを配布します。  
3. ファイルをダブルクリックしてブラウザ（ChromeやEdge）で開くだけで使えます。

## **📖 使い方（児童・生徒向け）**

1. **書く**: 「題名」「学年」「氏名」を入力し、本文を書きます。  
2. **保存する**:  
   * 自動でブラウザに保存されます。  
   * 「保存」ボタンを押すと、パソコンの中にファイル（.txt）として保存できます。提出用に使えます。  
3. **続きから書く**:  
   * 同じパソコンなら、また開くだけで続きから書けます。  
   * 「開く」ボタンから、保存したファイルを選んで読み込むこともできます。  
4. **印刷する**: 「印刷する」ボタンを押して、プリンターを選んで印刷するか、PDFとして保存します。

## **🛠️ 技術スタック**

* **HTML5 / CSS3 / JavaScript (ES6+)**: ランタイム不要、単一ファイル構成。  
* **Bootstrap 5**: UIフレームワーク。  
* **SweetAlert2**: 美しいダイアログ表示。  
* **Google Fonts**: Zen丸ゴシック（UI用）、Zenオールド明朝（原稿用紙用）。

## **⚠️ 外部サービスとの連携について**

このアプリは、GoogleドライブなどのクラウドストレージAPIとは直接連携しません（Lite版の設計思想のため）。

クラウドに保存したい場合は、以下の手順で行ってください。

1. 「保存」ボタンで端末に .txt ファイルをダウンロード。  
2. そのファイルを Googleドライブ や OneDrive に手動でアップロード。
